Redesigning This Website
Back as a slightly terrified first year, this grey website was the first thing
I ever coded outside of class, at a day long Women in Tech hackathon on campus.
And while I'm still a fan of this color scheme, this webiste definitely needs
a new design. Which is why it's great that, since my time as a first year, I've
learned that UX/UI is a thing and that it's helpful to prototype websites before
diving in and coding them! Unfortunatly, at the time of creating this, I didn't
have access to software such as Sketch or InDesign. But I was able to create a
prototype of what I (very roughly) want this website to look
like in the future using the web version of Adobe Spark.



My main priority is changing up the look of this site. Along the
way, I'd like to clean up how the code is written and structred
(for example, I now understand how to abstract away repetitive code,
something first-year-me didn't know). Now that I've gotten through
the first step of planning, I'm excited to redesign this website!
